# Roadmap - La Costa Sales Copilot

## Alto Impacto

- [x] **Dashboard com Pipeline Kanban**
  - [x] Pagina /dashboard com visao geral
  - [x] Colunas Kanban por status: Ativos | Remarketing | Fechados | Desqualificados
  - [x] Cards com inteligencia (temperatura, etapa, resumo, notas)
  - [x] Contadores por coluna
  - [x] Metricas resumidas no topo (total leads, conversao, follow-ups)
  - [x] Relatorio semanal inline (conversas novas, fechadas)
  - [x] Alertas de follow-up (conversas paradas 3+ dias)
  - [x] Drag-and-drop com @dnd-kit (arrastar cards entre colunas muda status)

- [x] **Follow-up Automatico**
  - [x] Detectar conversas paradas ha X dias

  - [x] Badge na sidebar "X pendentes"

- [x] **Simulador de Consorcio**

  - [x] Economia total destacada

  - [x] Conversas criadas no periodo

  - [x] Metricas semanais
